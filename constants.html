vunits.constants

Contains universal constants for catalysis research.

vunits.constants.F = <vunits.quantity.Quantity object>¶

Faraday’s constant (96485.33293056407 C/mol). Units supported have dimensions of charge/amount or equivalent quantities.

Type:Quantity
vunits.constants.G = <vunits.quantity.Quantity object>¶

Gravitational constant (6.67430e-11 m3 kg-1s-2). Units supported have dimensions of volume/mass/time or equivalent quantities.

Type:Quantity

Avogadro number (6.02214086e23 molecules/mol). This constant is dimensionless. Use this value instead of Avogadro’s constant to preserve units.
