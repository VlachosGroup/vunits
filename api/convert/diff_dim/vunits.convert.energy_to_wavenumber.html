vunits.convert.energy_to_wavenumber¶

vunits.convert.energy_to_wavenumber(energy, units_in='J', return_quantity=False, units_out='cm-1')¶

Converts energy to wavenumber

Parameters

  • energy (float) – Energy

  • units_in (str, optional) – Units corresponding to energy. Default is ‘J’.

  • return_quantity (bool, optional) – If True, returns Quantity. Otherwise, returns float.

  • units_out (str, optional) – Units corresponding to wavenumber. Default is ‘cm-1’.

Returns

wavenumber – Wavenumber corresponding to units_out.

Return type

float or Quantity obj
